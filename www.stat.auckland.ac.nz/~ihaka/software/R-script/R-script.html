R-script: R Transcript Recording

The script function implemented provided by this software provides an equivalent of the Unix/Linux script(1) command.

Typing script() at the R command prompt will cause a record to be kept of what takes places in the R session to kept in a file (default name “transcript.txt”). The recording is stopped by typing q().

Unlike the txtStart etc functions, this preserves the formatting of the lines the user types.
